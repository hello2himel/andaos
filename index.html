Contents

What ships. What doesn't.

Included
  • Hyprland + Waybar, pre-configured
  • Firefox with privacy defaults
  • PipeWire + Wireplumber audio
  • NetworkManager + applet
  • nano, vim, git, htop, btop
  • Full firmware & hardware support
  • Bluetooth with GUI controls
  • Archive support (zip, tar, 7z)
Not included
  • Office suites — install if needed
  • Dev environments — add your stack
  • Gaming platforms — AUR has them
  • Telemetry of any kind
  • Background services that do nothing
  • Software you'll never open

Installation

Three steps.

Write to USB
Step 01 · ~2 min

Download the ISO and flash it. Use dd on Linux/macOS, or Rufus on Windows.

# Linux / macOS sudo dd bs=4M if=andaos-*.iso of=/dev/sdX \ status=progress oflag=sync # Verify checksum sha256sum -c SHA256SUMS
Boot from USB
Step 02 · ~1 min

Restart and select the USB in the boot menu. Disable Secure Boot if needed. Live Hyprland desktop loads — no password.

Boot menu shortcut is usually F12, F11, or Del.
Install to disk
Step 03 · ~10 min · Standard Arch

Pure Arch underneath. Partition, pacstrap, configure. No magic installer — just the Arch way. Everything is transparent.

# 1. Partition your disk cfdisk /dev/sda # 2. Format and mount mkfs.ext4 /dev/sda1 && mount /dev/sda1 /mnt # 3. Install base system pacstrap /mnt base linux hyprland networkmanager # 4. Generate fstab and chroot genfstab -U /mnt >> /mnt/etc/fstab && arch-chroot /mnt
New to Arch? The official installation guide covers every step.

Audience

Not for everyone. That's the point.

Good fit
  • Arch users wanting a pre-configured start
  • Developers who need a clean workspace
  • Sysadmins deploying consistent environments
  • Students learning Linux from the ground up
  • Anyone who values knowing their system
Bad fit
  • Users expecting Ubuntu-style hand-holding
  • People who want everything pre-installed
  • Anyone uncomfortable with a terminal
  • Those needing commercial support
