install workflow

install in three simple windows

  1. Flash ISO to USB (dd / Rufus / Etcher).
  2. Boot into live AndaOS session.
  3. Follow standard Arch installation path.
# flash sudo dd bs=4M if=andaos-*.iso of=/dev/sdX status=progress oflag=sync # verify sha256sum -c SHA256SUMS # install base pacstrap /mnt base linux hyprland networkmanager
