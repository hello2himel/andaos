Installation

Three steps.

Write to USB
Step 01 · ~2 min

Download the ISO and flash it. Use dd on Linux/macOS, or Rufus / balenaEtcher on Windows.

# Linux / macOS sudo dd bs=4M if=andaos-*.iso of=/dev/sdX \ status=progress oflag=sync # Verify checksum sha256sum -c SHA256SUMS
Boot from USB
Step 02 · ~1 min

Restart your machine and select the USB drive in the boot menu. Disable Secure Boot if needed. The live XFCE desktop loads — no password required.

Boot menu shortcut is usually F12, F11, or Del depending on your firmware.
Install to disk
Step 03 · ~10 min · Standard Arch procedure

AndaOS is 100% standard Arch under the hood. Partition, pacstrap, configure. No custom installer, no magic scripts — just the Arch way.

# 1. Partition your disk cfdisk /dev/sda # 2. Format and mount mkfs.ext4 /dev/sda1 && mount /dev/sda1 /mnt # 3. Install base system pacstrap /mnt base linux xfce4 networkmanager # 4. Generate fstab and chroot genfstab -U /mnt >> /mnt/etc/fstab && arch-chroot /mnt
New to Arch? Follow the official installation guide — it covers every step in detail.

Audience

Not for everyone.
That's the point.

Good fit
  • Arch users who want a pre-configured starting point
  • Developers who need a clean, focused workspace
  • Sysadmins deploying consistent environments
  • Students learning Linux from the ground up
  • Anyone who values knowing what's on their machine
Bad fit
  • Users expecting Ubuntu-style hand-holding
  • People who want everything pre-installed
  • Anyone uncomfortable with a terminal
  • Those needing commercial support contracts
